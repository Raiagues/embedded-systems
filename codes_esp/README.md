### Set up the Environment Variables

`esp`

### Configure the Project

`idf.py set-target esp32`

### Combine bulding, flashing and monitoring 

`idf.py -p /dev/ttyUSB0 flash monitor`

### Exit IDF monitor

`Ctrl+]`
